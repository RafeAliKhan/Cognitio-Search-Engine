# COGNITIO-Search-Engine
## Cognitio | Python, HTML, CSS, JS, TypeScript | Live Demo
---
Cognitio is a next-generation conversational search engine driven by advanced artificial intelligence. Instead of delivering a list of blue links, Cognitio understands complex natural language queries, analyzes global data in real time, and provides direct, comprehensive answers with verified citations.

---

## 🚀 Features
Custom Web Crawler: Multi-threaded spider that traverses URLs with polite rate-limiting.<br>
Text Processing Pipeline: Automatic tokenization, stop-word removal, and Porter Stemming.<br>
Inverted Index: Highly optimized dictionary structure for fast term-to-document lookups.<br>
Relevance Ranking: Scores search results using standard [TF-IDF / Okapi BM25 / PageRank] algorithms.<br>
